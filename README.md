# Andriz Pray — Portfolio

Full-stack developer portfolio built with **Vue 3**, **TypeScript**, and **Tailwind CSS**.

**Live:** https://andrizpray.github.io/Portofolio/

## Tech Stack

- **Frontend:** Vue 3 (Composition API) + TypeScript
- **Styling:** Tailwind CSS v4
- **Build Tool:** Vite
- **Deployment:** GitHub Pages

## Features

- Dark blue/cyan theme — Server/DevOps monitoring vibe
- Real-time server stats widget (CPU, RAM, DISK)
- Skills with proficiency bars
- Projects showcase with GitHub stars
- Fully responsive design
- Smooth scroll navigation

## Local Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run deploy
```

## Project Structure

```
src/
├── App.vue          # Main app with all sections
├── main.ts          # Entry point
└── style.css        # Tailwind + custom styles
```

## Sections

1. **Hero** — Introduction with live server status widget
2. **About** — Bio and services
3. **Skills** — Tech stack with proficiency levels
4. **Projects** — Featured GitHub repositories
5. **Contact** — Email and social links
